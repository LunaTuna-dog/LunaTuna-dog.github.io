HTML REFRENCE SHEET

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<th> Table Header Define headings for columns or rows within the table.
<td> Table Data The table row container.
<tfoot> Table Footer The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Basic Html Tags

Tag Name Description
<!DOCTYPE> Document Type Defines the document type
<html> Hypertext Markup Language Defines a HTMl Document
<title> Title Defines a title for the document
<body> Body Defines the document's body
<head> Header Contains metadata/information for the document
<h1>-<h6> Headings 1-6 Represents 6 diffrent levels of section headings.
<p> Paraghraph Defines a Paraghraph
<a> Anchor Defines a hyperlink, which is used to link from one page to another
<br> Break Inserts a single line break
<!--..--> Comment Defines a comment
<img/> Image Defines an image
<figure> Figure Specifies self-contained content
<figcaption> Figure Caption Defines a caption for a <figure> element

Basic Html Attributes

Attribute Name Description
href Hypertext Refrence Indicates the destination of the hyperlink
style Style Used to add styles to an element
src Source Attribute specifies the path to the image to be displayed
width & height Width & Height specify the width and height of the image (in pixels).
